# 🌌 Space-A - 3D Solar System Animation

Space-A is a **3D Solar System project** built with **HTML, CSS, JavaScript, and Three.js**.  
It features interactive planets, mouse control, and educational planet information.




https://github.com/user-attachments/assets/c5ceea54-1bb2-4a83-8391-f56dba396628







## 🚀 Features

- 🌍 3D Solar System animation
- 🖱️ Mouse controls (rotate, zoom, drag)
- 🪐 Individual 3D planet models
- 📖 Planet information system
- 🎥 Smooth animations using Three.js
- 🌌 Space-themed UI design



## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Three.js



## 🎮 Controls

- **Mouse Drag** → Rotate scene / planet
- **Mouse Wheel** → Zoom in / out
- **Click** → Interact with planets (if enabled)

## 🌠 About

This project simulates a **realistic solar system environment** with interactive 3D planets and smooth camera movement using Three.js.

It is designed for learning, exploration, and visual experience of space.
